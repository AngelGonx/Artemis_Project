# Artemis_Project

Se utilizó Azure para la creación de un ChatBot amigable que orienta a los usuarios sobre dudas acerca del proyecto. Al ser un prototipo se agregaron preguntas generales pero conforme aparezcan más dudas se irá actualizando para que tendrá mayor repertorio de respuestas y de esa manera aclarar todas las dudas.

El programa que desarrollamos en Java trata de un prototipo para analizar el agua para detectar si requiere la aplicación de filtros cuyo objetivo es limpiarla y además el análisis del suelo detectando si es seguro para cuando se implemente la infraestructura del proyecto.
